# Matchstick
Epitech Project, Matchstick

## How to use Matchstick ?
* Clone this repository
* Use ``make`` into the folder
* (Optionnal : ``make clean`` to clean useless files)
* run ./matchstick x y (x / y = size of the rectangle)
* let's have fun !
